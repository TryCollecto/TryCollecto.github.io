Collecto

Open-Source, Self-Hosted Email Collection API

Collecto is an open-source, self-hosted email collection solution built for developers who care about control, scalability, and security.

In a world where hosted services often put your data at risk, Collecto provides a solution that you own and manage.

Whether you're building a high-traffic product or a small project, Collecto is production-ready out of the box, capable of handling everything from basic email signup forms to complex data workflows. Built with the backend developer in mind, Collecto defends against spam, DDoS attacks, and gives you the flexibility to integrate email signup functionality into any part of your stack.

How it Works

Creating a form and accepting signups is simple with Collecto.

# Create a signup form

POST /api/SignupForms
Body: {
    "formName": "Newsletter Signup"
}

# Accept signups

POST /api/EmailSignups
Body: {
    "formId": "1",
    "email": "user@example.com"
}
